# QA Manual Tests Portfolio

Этот репозиторий содержит мои мануальные тест‑кейсы, написанные в Qase.io для проекта Book Writer.

## Структура
- Book Management Suite — тесты для создания книг, глав, поиска и экспорта.
- Characters Suite — тесты для управления персонажами.
- World Suite — тесты для локаций и лора.
- Statistics Suite — тесты для подсчёта слов и today_words.
- Export Suite — тесты для выгрузки книги в разные форматы.

## Пример кейса
**DEMO-27 — Создание персонажа с пустым именем**  
Preconditions: книга открыта, раздел «Персонажи» доступен  
Steps:  
1. Открыть проект  
2. Перейти в раздел «Персонажи»  
3. Создать персонажа с пустым именем  
Expected: ошибка валидации
